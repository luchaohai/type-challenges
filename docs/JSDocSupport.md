# JSDoc Support

> 为了支持Javascript的库在Typescript中使用而做的支持。

## Step

1. Javascript库存在 JSDoc 语法的注释
2. 添加 TypeScript 到开发依赖项
4. 添加 tsconfig.json 配置 TypeScript
5. 运行 TypeScript 编译器生成 JS 文件对应的 d.ts 文件
